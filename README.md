# Ollama + ElevenLabs Voice Assistant

A local AI voice assistant combining:
- Ollama for text generation
- ElevenLabs for text-to-speech

## Setup
1. Install requirements: `pip install -r requirements.txt`
2. Add API keys to `.env`
3. Run: `python app/main.py`

## Features
- CLI interaction
- Markdown cleaning for natural speech
- Audio playback or saving
